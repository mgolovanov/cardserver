## Quick start
(1) Download and load this plugin javascript file after jQuery.  

```html
<script src="/js/jquery.tile.js"></script>
```

(2) Call `$('selector').tile()` in document ready event of jQuery.  

```javascript
$(function() {
  $('selector').tile();
});
```
